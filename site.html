Sobre HTML

No começo da minha formação em HTML, aprendi sobre a origem dos computadores e da internet, além de ter aprendido sobre conceitos de Server e Client e suas relações. Também aprendi sobre o que é HTML e como ele funciona, além de entender a importância do HTML para a construção de páginas web. HTML é uma linguagem de marcação que serve para estruturar o conteúdo da web, permitindo que os navegadores interpretem e exibam corretamente as informações contidas nas páginas.

O que são TAGS?

As TAGS são os elementos fundamentais do HTML, que definem a estrutura e o conteúdo das páginas web. Elas são compostas por uma palavra-chave entre sinais de menor e maior (< >), e podem ter atributos que fornecem informações adicionais sobre o elemento. As tags podem ser divididas em duas categorias: tags de abertura e tags de fechamento. As tags de abertura iniciam um elemento, enquanto as tags de fechamento encerram o elemento. Por exemplo, a tag <p> é usada para iniciar um parágrafo, enquanto a tag </p> é usada para encerrá-lo. Além disso, existem tags que não precisam de fechamento, como a tag <br>, que é usada para criar uma quebra de linha. As tags podem ser aninhadas, ou seja, uma tag pode conter outras tags dentro dela, criando uma hierarquia de elementos na página.

O que são Atributos?

Os atributos são informações adicionais que podem ser adicionadas às tags HTML para modificar seu comportamento ou aparência. Eles são definidos dentro da tag de abertura e consistem em um nome e um valor, separados por um sinal de igual (=). Por exemplo, a tag <a href="https://www.google.com"> usa o atributo href para especificar o endereço do link. Os atributos podem ser usados para definir propriedades como cor, tamanho, alinhamento e muito mais. Além disso, existem atributos globais que podem ser aplicados a qualquer tag HTML, como id, class e style. Esses atributos permitem que os desenvolvedores personalizem ainda mais o comportamento e a aparência dos elementos HTML.
